# Baskin Robbins Sales Dashboard

This project analyzes Baskin Robbins sales data using Power BI to generate business insights.

## Tools Used
Power BI
Excel
DAX

## Dashboard Insights
• Total Sales: 24.48M
• Total Orders: 49K
• Total Quantity Sold: 150K
• Average Order Value: 503.34

## Features
- Sales by Category
- Orders by Store
- Sales Trend by Year
- Payment Mode Analysis
- Customers by State

## Project Files
- Power BI Dashboard (.pbix)

- Dataset (.xlsx)
- Project Report (.pdf)
